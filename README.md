KidWise - Interactive Storytelling Platform

KidWise is an AI-powered Interactive Storytelling Platform designed to create engaging and immersive storytelling experiences for children. Users can generate AI-driven stories, illustrations, and voice narrations.

Features

Story Generation: AI-generated short stories based on user prompts.

Image Generation: AI-generated illustrations for each story segment.

Voice Narration: Text-to-speech conversion for an audio storytelling experience.

User Authentication: JWT-based authentication (AWS Cognito integration planned).

Frontend: React-based user interface with a seamless storytelling experience.

Tech Stack

Frontend: React, Material UI

Backend: Node.js, Express

AI Integration: Hugging Face (Story & Image Generation), AWS Polly (Text-to-Speech)

Deployment: AWS (Backend), Netlify (Frontend)
